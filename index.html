📋 Déroulement complet 10 séances
S1 — Lancement théâtral ⏱1h
Objectif : Plonger les élèves dans l'univers.
Modalité : Collectif.
Déroulement : • 5min Installation/Jauge • 15min Lecture prologue • 10min Échanges • 25min Création héros • 5min Bilan
S2 — Modelage & Mur de mots ⏱1h
Objectif : Enrichir le lexique.
Déroulement : • 5min Rappel • 15min Modelage récit d'aventure • 15min Mur de mots • 20min Vocabulaire • 5min Bilan
S3 — Chuchoteuses ⏱1h15
Objectif : Fabriquer l'outil de fluence.
Déroulement : • 10min Présentation • 35min Assemblage • 15min Déco • 15min Premier essai
S4 — Pratique guidée ⏱1h
Objectif : Amorcer le Chap.1.
Déroulement : • 5min Rappel • 15min Dictée à l'adulte • 10min Répartition groupes • 25min Début Chap.1 • 5min Bilan
S5 — Fluence + Chap.1 ⏱1h
Objectif : Installer le rituel fluence.
Déroulement : • 10min Fluence • 5min Consignes • 35min Écriture • 10min Lecture
S6 — Chap.1 suite ⏱1h
Objectif : Finaliser le 1er jet.
Déroulement : • 10min Fluence • 5min Rappel • 40min Écriture • 5min Bilan
S7 — Chap.1 fin + Mise en commun ⏱1h
Objectif : Terminer et harmoniser.
Déroulement : • 10min Fluence • 30min Fin écriture • 15min Lecture groupes • 5min Validation
S8 — Relecture & Correction ⏱1h
Objectif : Améliorer avec la grille.
Déroulement : • 10min Fluence • 10min Grille • 30min Relecture • 10min Réécriture
S9 — Mise au propre + Illustration ⏱1h
Objectif : Finaliser Chap.1 + Carte.
Déroulement : • 10min Fluence • 30min Recopie • 15min Illustration • 5min Carte
S10 — Clôture P1 ⏱1h
Objectif : Bilan du 1er cycle.
Déroulement : • 10min Fluence • 20min Lecture • 10min Jauge/Carte • 10min Bilan • 10min Annonce P2

📝 Mur de mots P1

👁️ Regarder

  • scruter, fixer, constater, réaliser

😨 Angoisse

  • frisson, panique, pétrifié, gorge nouée

🛠️ Construire

  • tresser, empiler, planter, consolider

🌙 La nuit

  • crépuscule, obscurité, refuge
🎯 Consignes 3 niveaux Chap.1
🔴 N1 Guidé
  • 5-7 phrases. Utilise scruter, pétrifié, ramasser
🟡 N2 Standard
  • 2 paragraphes. 3 verbes du Mur + 2 expressions d'angoisse
🟢 N3 Défi
  • 3 paragraphes. Dialogue ou pensée intérieure + présage
📖 Prologue
Le soleil se lève à peine, embrasant le ciel de nuances roses et orangées. Sur la plage, quatre silhouettes se glissent silencieusement vers la mer. Derrière eux, une pirogue de bois repose sur le sable. L'idée est venue la veille : aller passer la matinée sur l'îlot désert qui se dresse à l'horizon. Les parents refusent toujours qu'ils y aillent seuls. Tant pis, ce sera leur secret…
